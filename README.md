# Digital-literacy
**Project Overview**
This project focuses on developing essential digital literacy skills required for students in today’s digital world. As part of the role of a Student Digital Ambassador, the objective of this project is to help peers understand how to use digital tools effectively, communicate professionally, and stay safe online.The project is divided into five tasks, each covering an important aspect of digital literacy. It begins with creating an infographic to spread awareness about digital literacy concepts. It then involves building a digital portfolio using platforms like GitHub and LinkedIn to establish a professional online presence. The project also includes exploring coding and collaboration platforms such as HackerRank and Google Forms to enhance technical and practical skills.In addition, the project focuses on developing proper email communication skills and understanding online etiquette. Finally, it covers cybercrime awareness by studying phishing attacks and learning preventive measures.Overall, this project provides practical exposure to digital tools, improves online communication skills, and increases awareness about cybersecurity. It helps students become responsible and confident digital users, which is essential for both academic success and future career growth.
